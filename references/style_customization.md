# Word 段落 Runs 样式定制细节参考手册

在处理 Word 文档双语化时，如何在中文段落下方干净、优雅地追加英文翻译，且**绝对不破坏中文原有的字体大小、加粗、颜色及段落间距等排版样式**？

本文档将详细解密我们采用的 **Paragraph Run 级别样式注入方案**，并指导您如何利用命令行参数随心所欲定制英文外观。

---

## 🔬 Word 底层核心概念：段落（Paragraph）与文本块（Run）

在 Microsoft Word 的 XML（OpenXML）标准中，一个段落内部的文字是由一个个被称为 **Run**（文本块/文本运行）的标签构成的。

*   **段落 (Paragraph)**：代表一行或一个由回车划分的自然段。它决定了段落级别的样式（如：对齐方式、行间距、段前段后距、缩进等）。
*   **文本块 (Run)**：是段落内具有**完全相同字符样式**的连续文本片段。
    *   例如：在段落 `“请点击 确认 按钮”` 中，“确认”两字被加粗。那么底层 XML 会包含 3 个 Runs：
        1.  `Run 1`: "请点击 " (常规样式)
        2.  `Run 2`: "确认" (加粗样式)
        3.  `Run 3`: " 按钮" (常规样式)

### 💡 核心设计：为什么我们用 `add_run()` 实现换行？
如果我们直接修改段落的 text (`paragraph.text = new_text`)，Word 会**瞬间抹去该段落内部所有原有的样式和 Runs 标记**（例如局部的红字、粗体、下划线统统会丢失，退化为最普通的纯文本）！

为了绝对安全，我们使用了 `paragraph.add_run()` 方式：
```python
# 在段落的最后，追加一个全新的 run，在其内容开头加上 \n 实现“段内换行对照”
run = paragraph.add_run("\n" + translation)
```
*   **优势**：原中文段落的所有 Runs 及局部样式毫发无损，段落结构依然保持完整。追加的英文作为一个独立的 Run，只对英文部分的字体样式进行专门渲染。

## 🎨 英文 Runs 专属样式定制

由于英文部分是作为一个独立的 `Run` 存在，我们可以使用 `python-docx` 提供的 API，对其进行深度美化渲染：

### 1. 字号微调（使用 `Pt`）
```python
from docx.shared import Pt
# 英文通常比中文紧凑，缩小 1-2pt 可以让双语版层次更分明，避免排版臃肿
run.font.size = Pt(10) 
```

### 2. 颜色美化（使用 `RGBColor`）
```python
from docx.shared import RGBColor
# 默认使用优雅的暗蓝色（RGB: 31, 78, 121），能与中文黑色产生醒目的区分
run.font.color.rgb = RGBColor(31, 78, 121) 
```

### 3. 斜体强调 (`italic`)
```python
run.italic = True # 设置为斜体，在视觉上形成明显的“辅助阅读”感
```

## 🚀 高级玩法：利用参数一键微调双语排版

我们的 `inject.py` 已经对这些样式变量进行了参数化封装，支持在终端传入参数来自定义双语手册的质感。

以下是几套我们为您调试好的**高级排版参数方案**，您可直接复制运行：

### 方案 A：优雅学院风（默认推荐：暗蓝色 + 斜体 + 适当缩小）
> 视觉极具质感，中英主次对比极为强烈，商务手册首选。
```bash
python3 scripts/inject.py \
  --para-size 10.0 \
  --table-size 9.5 \
  --color "31,78,121"
```

### 方案 B：极简莫兰迪灰（深灰色 + 正常字号 + 斜体）
> 低调清爽，适合字数较少的极简操作指引。
```bash
python3 scripts/inject.py \
  --para-size 10.5 \
  --table-size 10.0 \
  --color "100,100,100"
```

### 方案 C：科技感浅青绿（科技暗青色 + 不设斜体 + 缩小）
> 适合带有强烈技术属性、代码指令的现代操作指南。
```bash
python3 scripts/inject.py \
  --para-size 9.5 \
  --table-size 9.0 \
  --color "0,128,128" \
  --no-italic
```
